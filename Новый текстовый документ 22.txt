Настройки/Функции на странице «Создать вид»

•	Имя представления отчета» → задает отображаемое имя представления отчета для ролей: FirmManager и Advisor.

o	Текстовое окно
o	Лимит символов = 255
o	Проверка: как только пользователь щелкнет за пределами текстового поля «Имя представления отчета», подтвердите количество символов в отображаемом имени <= 256
	Если проверка не пройдена, отобразить красную рамку вокруг текстового поля.

•	Компонент Radio Icon «Report Type» с опциями для Firm Report / Client Report → задает контекст просмотра Report View.

•	Компонент Radio Icon «Формат» с опциями для CSV/PDF/XLS → устанавливает формат просмотра отчета

•	Компонент Radio Icon «Ориентация» с параметрами «Пейзажная/Портретная» → устанавливает ориентацию представления отчета.
